# Intelligent Chatbot for Mental Health Support

## Description
This is a simple AI-based chatbot that provides emotional support using sentiment analysis.

## Features
- Detects user emotion (positive, negative, neutral, crisis)
- Provides supportive responses
- Runs in command prompt

## Setup
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

## Note
This is not a medical tool. It only provides basic emotional support.
